# FitRock App Icon 资源

## 文件说明

- `FitRock_Icon.svg` - 向量源文件，可在浏览器打开预览
- `FitRock_Icon.pdf` - 可导入 Xcode 的 PDF 版本

## 生成步骤

### 方法1: 使用 Safari 截图
1. 用 Safari 打开 `FitRock_Icon.svg`
2. 截图并保存为 PNG

### 方法2: 使用 macOS 预览
1. 用预览打开 SVG
2. 导出为 PNG

### 方法3: 使用 Xcode
1. 在 Xcode 中打开 Assets.xcassets
2. 选择 AppIcon
3. 拖入 1024x1024 PNG 图片

## 建议尺寸

| 尺寸 | 用途 |
|------|------|
| 1024x1024 | App Store |
| 180x180 | iPhone Notification (3x) |
| 120x120 | iPhone App (2x) |
| 87x87 | iPhone Spotlight (3x) |
| 80x80 | iPad App (2x) |
| 76x76 | iPad App |
| 58x58 | iPhone Settings (2x) |
| 40x40 | iPad Settings (2x) |
